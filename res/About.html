DUMB is a standalone Java application.
DUMB's main purpose is to fix PDFs exported with SMART notebook software.
Additionally, documents can be converted without fixing them.

Instructions

Operations

Initiating an operation is done as follows:

  1. Select the desired type of operation (Fix/Convert)
  2. Select the source file (Open). Note that the extension needs to match the file type of the document
  3. Select the destination file (Save). Again, the file extension has to match the desired file type
  4. Optionally tweak the preferences (Fixed Document Properties) to your desires (only relevant for fixing)
  5. Initiate the operation (Start)

Fixed Document Properties

Check the boxes to override document properties.

  • Pages: How to handle multiple pages - either creating a folder or merging them
  • Background: Allows tweaking the background - killing/filling or inside margins
  • Width: Total page width in centimetres (cm)
  • Margin: Margin of all edges in centimetres (cm)
  • Margin-Left/Right/Top/Bottom: Can override the respective margins
